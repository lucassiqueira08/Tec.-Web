Graduação em Banco de Dados


Objetivo

    Com o curso de Banco de Dados você estará preparado para
gestão e análise de tecnologias e dados, podendo atuar como
Administrador de Banco de Dados, Gerente de Servidores,
Analista de Informações e Arquiteto de Dados.


Sobre o Curso

    Com o curso de Banco de Dados você
será um profissional versátil e com habilidades
para atuar com diversas plataformas e estruturas de
desenvolvimento e administração de sistemas de banco de dados.

    Você terá todo repertório técnico e teórico,
conhecendo a importância da segurança de compartilhamento
de dados nas empresas modernas, dominando as melhores técnicas
e ferramentas de criação e implementação,
além de impulsionar a capacidade de gerência de equipes.


Grade Currícular

    • 1º Semestre
    • Comunicação e Expressão
      • Redija textos com os princípios de coesão e coerência, objetividade, concisão e clareza, dominando as regras gramaticais, localizando erros e corrigindo-os.
    • Fundamentos de BD
      • Domine os Fundamentos de Banco de Dados e a Modelagem Conceitual. Conheça o Esquema Lógico Relacional a partir do Esquema Conceitual, utilize a normalização de relações para elevar a qualidade do Projeto Lógico de BD.
    • Lógica de Programação
      • Você conhece a lógica de programação de computadores, dominando as estruturas básicas de controle de fluxo e de repetição, desenvolvendo a técnica de refinamentos sucessivos na solução de problemas, aprendendo a decompor um problema em sub-tarefas ou módulos. Conheça ainda as Estruturas de Dados na solução de problemas, desenvolvendo as estruturas de dados homogêneas: vetores e matrizes.
    • Matemática Aplicada
      • Conheça os conceitos fundamentais de lógica formal, dominando os métodos matemáticos e estatísticos para análise de conjuntos de dados. Saiba quais operações matemáticas devem ser realizadas para tratamento e representação de dados.
    • Linguagem de Programação I
      • Conheça as etapas de geração de um programa e domine as estruturas básicas de controle de fluxo na linguagem de programação Java. Desenvolva a técnica de refinamentos sucessivos na solução de problemas, aprendendo a decompor um problema em subtarefas ou módulos.
    • Introdução a Internet das Coisas - IoT
      • A Internet das Coisas é uma revolução tecnológica a fim de conectar dispositivos eletrônicos utilizados no dia-a-dia (como aparelhos eletrodomésticos, eletroportáteis, máquinas industriais, meios de transporte etc.)

    • 2º Semestre
    • Devops
      • O DevOps é a combinação de filosofias culturais, práticas e ferramentas que aumentam a capacidade de uma empresa de distribuir aplicativos e serviços em alta velocidade: otimizando e aperfeiçoando produtos em um ritmo mais rápido do que o das empresas que usam processos tradicionais de desenvolvimento de software e gerenciamento de infraestrutura
    • Analise Exploratoria de Dados
      • Fornecer ao aluno condições de organizar e descrever conjuntos de dados, dominar os fundamentos básicos de estatística descritiva,fazer planejamento de amostras e realizar testes estatísticos
    • Desenvolvimento de Sistemas
      • Desenvolva uma aplicação Web utilizando Java ServerPages, Servlets e Java ServerFaces. Compreenda os conceitos envolvidos no mapeamento objeto-relacional da API de persistência da especificação JEE e utilize uma implementação desta API. Aprenda a implementar web services utilizando a API JAX-WS.
    • Engenharia de Software
      • Engenharia de Software é uma área da computação voltada à especificação, desenvolvimento, manutenção e criação de sistemas de software, com a aplicação de tecnologias e práticas de gerência de projetos e outras disciplinas, visando organização, produtividade e qualidade.
    • Gestão de Projetos
      • A gestão de projetos é uma ferramenta essencial para garantir eficácia, orçamento, prazo e implementação com êxito de novos processos em grandes e pequenas empresas. Um bom gerenciamento de projetos envolve cuidado do início ao fim da sua gestão, se tornando um método muito valioso no mercado de trabalho.
    • Libras(optativa)
      • As Línguas de Sinais (LS) são línguas naturais das comunidades surdas. Ao contrário do que muitos imaginam as Línguas de Sinais não são simplesmente mímicas e gestos soltos, Utilizados pelos surdos para facilitar a comunicação.
    • Linguagem SQL
      • Conheça uma linguagem para definição e manipulação de dados em Sistemas de Banco de Dados. Domine os recursos da linguagem SQL, criando uma base sólida de consulta.
